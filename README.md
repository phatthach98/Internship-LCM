# Intern-LCM-PatientList
Simple LCM app created with Angular v7.2 and Ionic v4.0
## Technologies
Project is created with:
* Angular version: 7.2
* Ionic version: 4.x
* TypeScript version: 3.1
## Setup
Before running this project make sure having Nodejs.
To run this project, install it locally using npm:
* Step 1: npm install -g ionic
* Step 2: ionic start PatientList blank
* Step 3: Clone this project 
* Step 4: Paste this project into PatientList folder
